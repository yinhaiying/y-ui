

# 快速上手

## 说明
y-ui是基于Vue2.x的一套UI组件库，该组件库设计的目的是为了更好地熟练Vue相关API的使用，仅用于学习，
请勿在生成环境中使用。该组件库的实现参考了ElementUI，Ant Design等组件库的实现，承诺不用于商用，如有侵权，请联系删除。
## 安装


y-ui支持直接下载并用 `<script>` 标签引入，同时支持使用npm 和 yarn 安装。

**使用npm安装**
```javascript
npm install y-ui -S
```
**使用yarn安装**
```javascript
yarn add  y-ui -S
```

## 导入组件
**完整导入**

```javascript
import Vue from 'vue
import yUi from 'y-ui'
Vue.use(yUi)
```
